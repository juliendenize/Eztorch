eztorch.losses.compute_sce_token_masks(batch_size, num_tokens, num_negatives, positive_radius=0, keep_aligned_positive=True, use_keys=True, use_all_keys=False, rank=0, world_size=1, device='cpu')[source]¶

Precompute the masks for SCE with tokens.

Parameters:

  • batch_size (int) – The local batch size.

  • num_tokens (int) – The number of tokens per instance.

  • num_negatives (int) – The number of negatives besides the non-positive key elements.

  • positive_radius (int, optional) – The radius of adjacent tokens to consider as positives.

    Default: 0

  • keep_aligned_positive (bool, optional) – Whether to keep the aligned token as positive.

    Default: True

  • use_keys (bool, optional) – Whether to use the non-positive elements from the aligned key as negatives.

    Default: True

  • use_all_keys (bool, optional) – Whether to use the non-positive elements from all the gathered keys as negatives.

    Default: False

  • rank (int, optional) – Rank of the current process.

    Default: 0

  • world_size (int, optional) – Number of processes that perform training.

    Default: 1

  • device (Any, optional) – Device that performs training.

    Default: 'cpu'

Return type:

Tuple[Tensor, Tensor, Tensor, Tensor, Tensor]

Returns:

  • Mask to ignore similarities for the query.

  • Mask to ignore similarities for the keys.

  • Mask of positives for the query.

  • Mask to keep log values.

  • The number of positives for the specific token.

Spot loss¶

eztorch.losses.compute_spot_loss_fn(class_preds, class_target, has_label, ignore_class, class_weights=None, mixup_weights=None, class_loss_type=LossType.BCE, alpha=0.25, gamma=2)[source]¶

Compute the soccernet loss function which is a classification loss.

Parameters:

  • class_preds (Tensor) – Predictions for the classes. Expected shape: (B, T, C).

  • class_target (Tensor) – Multi-label encoded class. Can be continuous label in case of mixup. Expected shape: (B, T, C).

  • has_label (Tensor) – Whether there is a label. Expected shape: (B’, T, C).

  • ignore_class (Tensor) – Whether to ignore class. Expected shape: (B’, T, C).

  • class_weights (Optional[Tensor], optional) – Weights of negatives and positives for BCE loss. Expected shape: (2, C).

    Default: None

  • mixup_weights (Optional[Tensor], optional) – Weights for mixup for loss. Expected shape: (B’, T, C).

    Default: None

  • class_loss_type (LossType, optional) – Type of loss to use. Can be BCE, softmax or Focal.

    Default: LossType.BCE

  • alpha (float, optional) – For focal loss.

    Default: 0.25

  • gamma (float, optional) – For focal loss.

    Default: 2

Return type:

Tensor

Returns:

The reduced sum of the classification losses.
